# Tests Context

**Purpose:**
This directory contains the automated Jest test suite for the backend application and the scraper.

**Key Files:**
- `server.test.js`: Tests for the Express API endpoints defined in `server.js`.
- `run-scraper.test.js`: Tests validating the Puppeteer scraping functionality.

**How AI should use this folder:**
When modifying `server.js` or scraper scripts, you must ensure existing tests pass and write new tests here for new features or bugfixes.
